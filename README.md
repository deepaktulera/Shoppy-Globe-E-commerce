# Shoppy Globe E-Commerce

Shoppy Globe is a simple e-commerce application built with React, Redux Toolkit, React Router, and Tailwind CSS.

## Project Structure

```text
shoppy-globe-e-commerce/
│
├── shoppy-globe-frontend/
│   ├── src/
│   ├── public/
│   ├── package.json
│   └── ...
│
└── README.md
```

## Frontend Application

The complete frontend application is located inside:

```text
shoppy-globe-frontend
```

## Features

* Product Listing
* Product Search
* Product Details Page
* Shopping Cart
* Quantity Management
* Checkout Page
* Responsive Design
* Local Storage Support
* Toast Notifications

## Getting Started

Navigate to the frontend folder:

```bash
cd "Shoppy-Globe-Frontend"
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## Tech Stack

* React
* Redux Toolkit
* React Router DOM
* Tailwind CSS
* React Toastify
* Lucide React
