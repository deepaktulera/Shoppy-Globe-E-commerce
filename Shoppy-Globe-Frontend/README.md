# Shoppy Globe

Shoppy Globe is a simple e-commerce web application built using React, Redux Toolkit, React Router, and Tailwind CSS. Users can browse products, search products, view product details, add items to the cart, manage quantities, and place orders through a checkout page.

## Repository

GitHub: https://github.com/deepaktulera/Shoppy-Globe-E-commerce

## Features

* Browse products
* Search products by name
* View product details
* Add products to cart
* Increase or decrease product quantity
* Remove products from cart
* Cart data stored in Local Storage
* Checkout page
* Responsive design for mobile and desktop
* Toast notifications for cart actions

## Tech Stack

* React
* Redux Toolkit
* React Router DOM
* Tailwind CSS
* React Toastify
* Lucide React

## Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project folder

```bash
cd "Shoppy-Globe-Frontend"
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```
